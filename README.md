# ForgeHelm
Testing out Rust for API Architecture

The purpose of this project is to create a learning experience for a full stack application built with JS (React), Rust (Actix-Web), and PostgreSQL.
The main focus of this application is to serve as an interface for creating and managing multiple real world projects through individual and merged grids and gantt charts.

Main Features to be implemented:
- Dynamic Grid Component for Individual Schedules
- Dynamic Gantt Chart Component for Individual Schedules
- Dynamic Group Grid Component for List of Schedules
- Dynamic Group Gantt Component for List of Schedules
- Dynamic dependencies between as many custom fields as possible
- Automation Rules using triggers on custom fields
- User Roles & Permissions
